Simple plugin to wrap Kontera zone tags around content and/or comments

== Description ==
We can explicitly control what parts of our sites to display Kontera's ContentLink ads by using "ad zone" div tag as follows.

`<div class="KonaBody">`
`...Content we want ad links appear here...`
`</div>`

This Wordpress plugin can help wrapping Kontera's ad zone tags around post content and/or comments easily.

== Installation ==

1. Upload `kontera_ad_wrap.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Options - Kontera Ad Wrap' menu for selecting options
